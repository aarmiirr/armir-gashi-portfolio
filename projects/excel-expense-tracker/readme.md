# Excel Expense Tracker with Monthly Budget

## Project Overview

A comprehensive expense tracking system built in Microsoft Excel that helps monitor spending across 6 categories with automated budget alerts and visual dashboards.

##  Features

- **Budget vs Actual Comparison:** Real-time tracking of spending against monthly budgets
- **Category Management:** 6 spending categories (Food, Transportation, Entertainment, Utilities, Shopping, Other)
- **Automated Calculations:** SUMIF formulas calculate category totals automatically
- **Budget Alerts:** Conditional formatting provides color-coded warnings for over-budget categories
- **Visual Dashboard:** Pie chart showing spending distribution across categories
- **Trend Analysis:** Monthly spending patterns (achieved 15% reduction in entertainment costs)

## Excel Skills Demonstrated

- **Formulas:** SUMIF, IF statements, basic arithmetic
- **Data Validation:** Dropdown lists for categories
- **Conditional Formatting:** Color-coded budget status (Green: Under budget, Yellow: Near limit, Red: Over budget)
- **Charts:** Pie charts for spending visualization
- **Cell References:** Absolute and relative references
- **Data Organization:** Structured tables for efficient data entry

## How to Use

1. Download `expense-tracker.xlsx`
2. Open in Microsoft Excel (2016 or later recommended)
3. Enter your monthly budget amounts in the Budget column
4. Add expenses with date, category, and amount
5. Watch automatic calculations and alerts update in real-time
